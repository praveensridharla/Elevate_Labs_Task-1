# Java Console Calculator

This project is a simple calculator application developed using Java. It runs in the command line and allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator continues running until the user chooses to exit, making it convenient for performing multiple calculations in a single session.

## About the Project

I created this project to strengthen my understanding of Java fundamentals and gain practical experience with building interactive console applications. While developing it, I focused on writing clean code, handling user input properly, and managing common errors such as invalid entries and division by zero.

## Features

* Perform addition, subtraction, multiplication, and division
* Interactive menu-driven interface
* Continuous execution until the user exits the program
* Handles division-by-zero errors safely
* Validates user input to prevent unexpected crashes
* Simple and easy-to-use command-line experience

## How to Run

### Requirements

* Java JDK installed on your system
* Any code editor or Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.)

### Compile the Program

```bash
javac Java_Console_Calculator.java
```

### Execute the Program

```bash
java Java_Console_Calculator
```
